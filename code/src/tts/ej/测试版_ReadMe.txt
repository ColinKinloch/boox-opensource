请注意：此版本为测试版，仅供测试使用。

inc目录
C/C++ API包含文件，文件扩展名常以h,hxx,inl表示，此文件定义了基于本产品开发所使用的函数接口和一些常量定义，此目录中的文件供开发人员使用。

data目录
产品SDK运行时刻所使用的数据文件，文件扩展名常以.dat表示（若数据文件以数据的方式表示，则此文件的扩展名为.c或cpp），此数据文件是供各个产品的引擎使用，用户不直接操作这些文件，此目录中的文件在开发阶段和程序运行阶段使用。

lib目录
位于此目录中的文件为开发人员使用的静态或动态库文件，文件扩展名常为lib,a或dll，静态库在编译时刻供开发人员使用，动态库在运行时刻使用。

注意：data文件和lib文件在使用之前应去掉文件名最后一个下划线之前内容（最后一个下划线也应去掉），如：文件名为G729_SmallFile_标C_li_eVC4_WinCE42_SMARTPHONE_ARMV4_ejTTS.dll 改为ejTTS.dll，音库文件的修改方法也如此。

doc目录
此目录中的文件为产品的帮助文档，包含产品开发手册，技术白皮书等。

sample目录
此目录中的文件为可以运行的执行程序及播放层的源代码，基于本SDK开发的一个应用程序，演示产品的主要功能特性，源代码可供开发人员作为程序设计时的参考或直接引用。
注意：只提供pc下的或WinCE系统下的可执行文件。

test.c文件
此文件为基于本产品SDK开发的一个简单应用。
